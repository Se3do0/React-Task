# Animated Slide Showcase

This repository contains an example project showcasing animated slides built with Vite, React, TypeScript, Tailwind CSS, and shadcn-ui.

## Run locally

Requirements: Node.js and npm installed.

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

Build the production artifacts and deploy them using your preferred static host:

```sh
npm run build
npm run preview
```
